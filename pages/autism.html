Project Overview

Intelligent Speech Analysis

The AI-Powered ASD Facial Recognition is designed to automatically analyze facial images and classify fluent and stuttered speech using deep learning.

The application combines a React frontend, FastAPI backend, OpenCV-based preprocessing, and a fine-tuned Vision Transformer (ViT) model to provide reliable speech fluency assessment.

The project demonstrates modern AI deployment by integrating machine learning with scalable web technologies and an intuitive user interface.
