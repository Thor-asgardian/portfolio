Project Overview

Intelligent Healthcare Assistance

The AI-Powered Bilingual Healthcare Assistant was developed to provide multilingual healthcare guidance through conversational artificial intelligence. It enables users to communicate using voice or text while receiving context-aware responses based on trusted healthcare knowledge.

The application integrates speech recognition, semantic search, Retrieval-Augmented Generation, vector databases, and Large Language Models to deliver reliable medical assistance while reducing hallucinations commonly associated with standalone language models.

Built using Flask, LangChain, FAISS and modern AI technologies, the project demonstrates how Generative AI can be applied to improve healthcare accessibility across multiple languages.
